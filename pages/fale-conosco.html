Perguntas Frequentes

Quando o FinanFun estará disponível?

Estamos trabalhando no desenvolvimento do aplicativo. Em breve divulgaremos mais informações sobre o lançamento.

Para que idade o app é indicado?

O FinanFun foi desenvolvido para crianças e adolescentes de 6 a 18 anos, com conteúdo adaptado para cada faixa etária.

O FinanFun é seguro para as crianças?

A segurança é uma das nossas principais prioridades. Os pais têm controle total para estabelecer funcionalidades e limites na plataforma. Nosso sistema de proteção de dados segue rigorosamente a Lei Geral de Proteção de Dados (LGPD).

Como funciona o sistema de recompensas?

As crianças acumulam pontos ao completar missões e desafios educativos. Esses pontos podem ser trocados por recompensas virtuais ou combinados com os pais para recompensas reais, incentivando o aprendizado contínuo.

Os pais podem acompanhar o progresso?

Sim! Oferecemos um painel completo para que os pais acompanhem o desenvolvimento dos filhos, visualizem atividades realizadas e ajustem configurações conforme necessário.

Quanto custa o FinanFun?

Ainda estamos definindo os planos de assinatura. Nosso objetivo é oferecer opções acessíveis para que todas as famílias possam proporcionar educação financeira de qualidade para seus filhos.
