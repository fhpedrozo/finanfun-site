Perguntas Frequentes

Quando o FinanFun estará disponível?

Estamos trabalhando no desenvolvimento do aplicativo. Em breve divulgaremos mais informações sobre o lançamento.

Para que idade o app é indicado?

O FinanFun foi desenvolvido para crianças e adolescentes de 6 a 18 anos, com conteúdo adaptado para cada faixa etária.

O FinanFun é seguro para as crianças?

A segurança é uma das nossas principais prioridades. Os pais têm controle total para estabelecer funcionalidades e limites na plataforma. Nosso sistema de proteção de dados segue rigorosamente a Lei Geral de Proteção de Dados (LGPD) brasileira.

Este app é gratuito?

As funcionalidades básicas de conta corrente são gratuitas. Para funcionalidades mais específicas e avançadas, oferecemos planos com custo acessível adaptados para cada necessidade familiar.

Como posso me tornar um parceiro?
